# NGI Plate database

A tool for keeping track or plates used at NGI. 

## Location barcodes

Storage units (e.g. freezers and fridges), racks and positions within racks all have specific ID's described below:

Full position barcode
`R[NNNN]X[NN]Y[NN]`

Rack location only
`R[NNNN]_`

Storage unit only
`S[NNNN]`

	S: Storage unit
	R: Rack
	X/Y: position
